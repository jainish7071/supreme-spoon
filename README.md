# Supreme Spoon

A prototype tool for AI-generated Instagram Reels and YouTube Shorts.

## Setup

1. Install Python dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Generate a sample video:

   ```bash
   python -m video_tool --topic "AI daily news"
   ```

## Notes

- If you see an import error for `moviepy`, install the dependencies again.
- This first step uses local fallback video generation and will be replaced with AI model integration next.
